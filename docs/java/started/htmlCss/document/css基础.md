# CSS基础

## CSS样式分类

### 内部样式

```css
<style type="text/css">
    div {
        width: 100px;
    }
</style>
```

### 行内样式

```css
<div style="width: 200px;"></div>
```

### 外部样式

```css
<link type="text/css" rel="stylesheet" href="a.css">
```


## CSS基础选择器

### 标签选择器

```css
span {
    font-size: 100px;
}
```

### 类选择器

```css
.blue {
    color: blue;
}
```
