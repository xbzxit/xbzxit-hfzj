# 多数据源

## 创建数据库

```sql
CREATE DATABASE `mp1` /*!40100 DEFAULT CHARACTER SET utf8mb4 */;
use `mp1`;
CREATE TABLE product
(
 id BIGINT(20) NOT NULL COMMENT '主键ID',
 name VARCHAR(30) NULL DEFAULT NULL COMMENT '商品名称',
 price INT(11) DEFAULT 0 COMMENT '价格',
 version INT(11) DEFAULT 0 COMMENT '乐观锁版本号',
 PRIMARY KEY (id)
);


INSERT INTO product (id, NAME, price) VALUES (1, '外星人笔记本', 100);

```


## 导入依赖

```yaml
<dependency>
  <groupId>com.baomidou</groupId>
  <artifactId>dynamic-datasource-spring-boot-starter</artifactId>
  <version>3.5.0</version>
</dependency>
```

## 配置数据源

```yaml
spring:
 # 配置数据源信息
datasource:
 dynamic:
   # 设置默认的数据源或者数据源组,默认值即为master
  primary: master
   # 严格匹配数据源,默认false.true未匹配到指定数据源时抛异常,false使用默认数据源
  strict: false
  datasource:
   master:
    url: jdbc:mysql://10.105.0.235:3306/mp?characterEncoding=utf-8&useSSL=false
    driver-class-name: com.mysql.cj.jdbc.Driver
    username: root
    password: 
   slave_1:
    url: jdbc:mysql://10.105.0.235:3306/mp1?characterEncoding=utf-8&useSSL=false
    driver-class-name: com.mysql.cj.jdbc.Driver
    username: root
    password: 
```


## 用户业务层

### 业务接口

```java
public interface UserService extends IService<User> {
}
```

### 业务实现类

```java
@DS("master") //指定所操作的数据源
@Service
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements
UserService {
}
```


## 商品业务层

### 业务接口

```java
public interface ProductService extends IService<Product> {
}
```

### 业务实现类

```java
@DS("slave_1")
@Service
public class ProductServiceImpl extends ServiceImpl<ProductMapper, Product>
implements ProductService {
}
```
