# 集合常见操作

## List的常用操作

</br></br></br>
* **获取某个属性**

> 获取所有用户的姓名集合
>
> ```java
> List<User> userList = new ArrayList();
> [
>    {"username":"zhangsan","age",22},
>    {"username":"lisi","age",21}
> ]
>
> ```

```java
List<String> nameList = userList.stream().map(User :: getName).collect(Collectors.toList());
```
</br></br></br>

* **获取某个属性去重**

> 获取所有用户都分布在哪些城市
>
> - 数据
>
> ```java
> List<User> userList = new ArrayList();
> [
>    {"username":"zhangsan","age":22,"address":"北京"},
>    {"username":"lisi","age":21,"address":"北京"}
> ]
> ```

```java
List<String> addressList = userList.stream().map(User :: getName).distinct().collect(Collectors.toList());
```
</br></br></br>
