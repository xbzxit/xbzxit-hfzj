# 禅道

## 禅道项目管理流程

> 产品经理创建产品，并整理产品的需求
>
> 项目经理创建项目，并整理项目的需求 （产品与项目进行关联）
>
> 项目开始由开发团队和测试团队配合完成
>
> - 开发
>   1. 开发团队开会决定，对开发任务进行安排并把任务指派给开发人员
>   2. 开发领取开发任务并估算任务的开发周期
> - 测试
