# Jmeter使用示例

## 第一个案例

> 需求： 使用JMeter访问百度首页接口，并查看请求和响应信息

```bash
1. 启动JMeter

2. 在【测试计划】下添加【线程组】

3. 在【线程组】下添加【HTTP请求】 取样器

4. 填写【HTTP请求】的相关请求参数

5. 在【线程组】添加【察看结果树】监听器

6. 点击【启动】按钮运行，并查看结果
```
